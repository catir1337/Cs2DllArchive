# CS2 DLL Archive

Автоматический архив бинарников (DLL) Counter-Strike 2, обновляющийся при выходе каждого нового патча Valve.

## Как это работает

GitHub Actions каждые 30 минут:
1. Скачивает через [DepotDownloader](https://github.com/SteamRE/DepotDownloader) актуальные версии нужных DLL из публичного депо CS2 (appid `730`), анонимным логином, без Steam-аккаунта.
2. Сравнивает их с последней сохранённой версией в репозитории.
3. Если файлы изменились (то есть вышло обновление игры) — создаёт новую папку с датой и коммитит в неё скачанные файлы.
4. Если изменений нет — ничего не делает.

Задержка обнаружения нового патча — не больше 30 минут.
